# Zelin - 泽聚万象 · 麟启新生

泽麟是由 JuCoin 生态构建的合规数字资产平台。致力于通过技术创新，将文化艺术与数字经济深度融合，为用户提供安全、透明、有价值的收藏体验。

本仓库包含了泽麟官方展示页面的前端代码（基于 Vite + React）。

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
